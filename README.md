# UE5-UE4-PetAndInventorySystemTemplate

Pet System & Item Inventory System Template For UE4.27 & UE5.5 - Technically you can create a Pokémon-like game from this pet system.

The project was originally created with UE4.21 but has been tested it can easily be converted to UE4.27 & UE5.5 and works.

Important note for UE5.5: DO NOT create children of pet_parent_BP and Item_Master_BP but instead create a duplicate of an already made child of those BP Parents, That is the only work around to make these Pet and Item invtory systems work on UE5
